# p5sbankeyboard

[全てあなたの所為です。](https://www.youtube.com/channel/UC27m3R6W5sP7-QLfMTEso2w)『[教育](https://www.youtube.com/watch?v=nNeOqvtS39c)』風鍵盤プレイグラウンド

## 操作

- **右矢印**: 基準を 1 オクターブ上げる
- **左矢印**: 基準を 1 オクターブ下げる
- **F**: 鍵盤切り替え
- **D**: 複合モード切り替え
- **R**: 復号結果をリセット

> [!WARNING]
> MIDI 入力が必要です。MIDI 入力できる実機がない人はバーチャル MIDI ケーブルを用いるのがおすすめ。
